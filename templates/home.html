This is the home page

  • Clone this repository
  • Create a branch using your name
  • Create an HTML file in the templates folder that has your name in an H1 element.
  • Add a link on the home/index.html page that links to the page that has your name
  • Add a route in the config that routes the ng-view to your page
  • Push your branch to the repo
  • Create a pull request
